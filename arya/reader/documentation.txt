Usage for module: from reader import *

Player (EX: p1)
- p1.x = Player 1's x position
- p1.y
- p1.dmg = Player 1's percent damage
- p1.dist(p2) = Calculates the distance between two players
- p1.stocks = How many lives Player 1 has
- p1.facing = Says if Player 1 is facing "left" or "right"
- p1.action = What state the player is (just refer to state dictionary in mmap)
- p1.invulnerable = Bool
- p1.action_frame = How many frames has Player 1 been in an action
- pprint -> x, y, %, facing, action, action_frame, invulnerable
